# Versatile CAPTCHA Generation Using Machine Learning and Image Processing

## Overview

Versatile CAPTCHA Generation Using Machine Learning and Image Processing is a web-based security system designed to protect applications from automated bot attacks. The project generates dynamic CAPTCHA images using advanced image processing techniques and machine learning, making them easy for humans to solve while difficult for automated programs to recognize.

Unlike traditional text-based CAPTCHAs, this system creates multiple CAPTCHA styles by applying random distortions, background patterns, noise, rotations, color variations, and object overlays. A Convolutional Neural Network (CNN) is integrated to analyze CAPTCHA complexity and improve the robustness of the generated challenges.

The application includes user registration and login with CAPTCHA verification, ensuring that only legitimate users can access the system. Every CAPTCHA is generated uniquely in real time, enhancing security and reducing the risk of automated attacks.
